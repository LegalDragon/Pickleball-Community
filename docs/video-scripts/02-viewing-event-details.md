# Video Script: Viewing Event Details

**Duration:** ~2 minutes
**Target Audience:** Users who found an event and want to learn more

---

## Introduction (10 seconds)

**Narration:**
"Let's explore how to view event details and understand what you're signing up for."

---

## Step 1: Open Event Details Modal (10 seconds)

**Action:** Click "View Details" on any event card

**Screenshot:** Event details modal opening with header visible

**Narration:**
"After clicking View Details, a modal opens showing everything about the event."

---

## Step 2: Event Header (20 seconds)

**Action:** Point to the header section

**Screenshot:** Event header showing: poster image, event name, date range, registration count

**Narration:**
"At the top you'll see:
- The event poster or banner image
- Event name and type badge
- Start and end dates
- Total registered teams/pairs and players
- Registration status - whether it's open, closing soon, or closed"

---

## Step 3: Details Tab Overview (25 seconds)

**Action:** Show the Details tab content

**Screenshot:** Details tab showing time, fee, divisions count, and description

**Narration:**
"The Details tab shows:
- **Event time** - When it starts
- **Entry fee** - Cost per person or per team
- **Number of divisions** - Different skill levels or age groups available
- **Event description** - Rules, format, and other important info
- **Venue location** - Click to see it on the map
- **Contact info** - Who to reach out to with questions"

---

## Step 4: The Register Button (15 seconds)

**Action:** Point to the prominent Register button

**Screenshot:** Register Now button (orange) prominently displayed

**Narration:**
"The big orange 'Register Now' button takes you to the Registrations tab where you can sign up. If you're already registered, this becomes a green 'View My Registrations' button."

---

## Step 5: Registrations Tab (30 seconds)

**Action:** Click on "Registrations" tab, then select a division from dropdown

**Screenshot 1:** Registrations tab with division dropdown
**Screenshot 2:** Division selected showing registered pairs/teams list

**Narration:**
"Click the Registrations tab to see available divisions and who's already signed up.

Use the dropdown to select a division. You'll see:
- Division name and description
- The Register button if you can sign up
- A list of all registered players, pairs, or teams
- Complete registrations show a green checkmark
- Those looking for partners show an amber 'Looking' badge"

---

## Step 6: Understanding Division Types (20 seconds)

**Action:** Show different division examples

**Screenshot:** Division info showing format (Singles/Doubles/Team), skill level, gender

**Narration:**
"Each division specifies:
- **Format** - Singles, Doubles, or Team
- **Skill level** - Like 3.0, 3.5, 4.0, etc.
- **Gender** - Men's, Women's, Mixed, or Open
- **Age group** - If applicable"

---

## Step 7: Checking Registration Status (15 seconds)

**Action:** Show the registration status indicators

**Screenshot:** Status badge showing "Registration Open" or "Closes in X days"

**Narration:**
"Always check the registration status at the top. It tells you if registration is open, when it closes, or if it's already full."

---

## Closing (10 seconds)

**Narration:**
"Now you know how to explore event details. Next, let's walk through the actual registration process."

---

1. Event details modal just opened (header visible)

3. Details tab content - full view
4. Register Now button (orange, prominent)
5. View My Registrations button (green, for registered users)
6. Registrations tab with division dropdown
