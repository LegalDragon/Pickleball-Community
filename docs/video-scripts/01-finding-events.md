# Video Script: Finding and Browsing Events

**Duration:** ~2 minutes
**Target Audience:** New users looking for pickleball events

---

## Introduction (10 seconds)

**Narration:**
"In this video, you'll learn how to find pickleball events near you on Pickleball Community."

---

## Step 1: Navigate to Events Page (15 seconds)

**Action:** Click "Events" in the bottom navigation bar (mobile) or top navigation (desktop)

**Screenshot:** Bottom navigation with Events icon highlighted

**Narration:**
"From anywhere in the app, tap the Events icon in the navigation bar. This takes you to the Events page where you can browse all available tournaments, open play sessions, and community events."

---

## Step 2: Understanding the Events Page Layout (20 seconds)

**Action:** Show the Events page with tabs visible

**Screenshot:** Events page showing "My Events" and "Browse" tabs

**Narration:**
"The Events page has two main tabs:
- **My Events** shows events you're registered for or organizing
- **Browse** lets you search for new events to join

If you're logged in, you'll start on My Events. Let's switch to Browse to find new events."

---

## Step 3: Browse Events Tab (15 seconds)

**Action:** Click on "Browse" tab

**Screenshot:** Browse tab with search filters and event list

**Narration:**
"The Browse tab shows upcoming events. You can see each event's name, date, location, how many players are registered, and the entry fee."

---

## Step 4: Using Location-Based Search (25 seconds)

**Action:** Show the location permission prompt and radius slider

**Screenshot 1:** Location permission prompt
**Screenshot 2:** Events sorted by distance with "X miles away" badges

**Narration:**
"For the best experience, allow location access. This sorts events by distance so you see the closest ones first. Each event card shows how many miles away the venue is.

You can also adjust the search radius using the slider to find events within 25, 50, 100 miles or more."

---

## Step 5: Filtering Events (25 seconds)

**Action:** Show the filter options - event type dropdown, location filters

**Screenshot 1:** Event type filter dropdown (Tournament, Open Play, Clinic, etc.)
**Screenshot 2:** State/City filter dropdowns

**Narration:**
"Use the filters to narrow your search:
- **Event Type** - Filter by tournaments, open play, clinics, or social events
- **Location** - Select a specific state or city
- **Search box** - Type keywords to find specific events by name"

---

## Step 6: Viewing Event Cards (20 seconds)

**Action:** Point to different parts of an event card

**Screenshot:** Single event card with annotations pointing to: event type badge, name, date, venue, registration count, fee

**Narration:**
"Each event card shows key information at a glance:
- The colored badge shows the event type
- Event name and date
- Venue name with a link to view the location
- Number of registered teams or pairs and total players
- Entry fee, if any"

---

## Step 7: Opening Event Details (10 seconds)

**Action:** Click "View Details" button on an event card

**Screenshot:** Event card with "View Details" button highlighted

**Narration:**
"Click 'View Details' to see full information about the event and register."

---

## Closing (10 seconds)

**Narration:**
"That's how you find events! In the next video, we'll show you how to view event details and register."
