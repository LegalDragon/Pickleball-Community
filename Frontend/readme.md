# Pickleball.College Client

This is the React frontend for Pickleball.College.

## Setup Instructions

1. Navigate to this directory
2. Run 
pm install to install dependencies
3. Run 
pm start to start the development server
4. The application will open at http://localhost:3000

## Available Scripts

- 
pm start - Runs the app in development mode
- 
pm run build - Builds the app for production
- 
pm test - Launches the test runner
- 
pm run eject - Ejects from Create React App

## Project Structure

- src/components - React components
- src/services - API service classes
- src/utils - Utility functions
- src/styles - CSS and styling files
